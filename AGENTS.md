# AGENTS.md

Reglas para cambios de baseline/docgen en este repositorio:

1. No agregar dependencias nuevas sin justificar explícitamente en el PR.
2. Limitar cambios a `supabase/functions/**` y `docs/docgen/**`, salvo instrucción directa del usuario.
3. Siempre entregar output con:
   - Archivos tocados.
   - Checklist de validaciones ejecutadas.
